# Shared

Shared modules contain reusable building blocks with no single feature ownership.

Ownership map:

- `shared/ui`: branded primitives, control contracts, modal and form shells
- `shared/lib`: pure helpers, adapters, formatting/error helpers, schema compatibility helpers
- `shared/auth`: auth utilities that are not feature-specific
- `shared/supabase`: shared Supabase-facing contracts or wrappers
- `shared/types`: cross-feature shared type helpers
